# PSH Summit 2026 — DevSecOps A–Z

Demo repository for the **PowerShell & DevOps Summit Bellevue 2026** conference session:

> **"DevSecOps Alphabet Soup — A–Z of Securing the Software Delivery Pipeline"**

## About This Repo

This repo contains all demo scripts, reference guides, and supporting material used during the live session. The session runs 80 minutes and walks through DevSecOps concepts and tooling from A to Z using GitHub, GitHub Advanced Security (GHAS), GitHub Copilot, and Azure.

**Audience:** Developers and DevOps engineers new to or deepening their DevSecOps knowledge.

## Demo Environment

- **Platform:** GitHub (personal account) + GitHub Advanced Security
- **AI Tooling:** GitHub Copilot (Autofix, code review, secret scanning)
- **Cloud:** Microsoft Azure
- **Session length:** 80 minutes

## Topics Covered (A–Z)

A subset of topics covered across the session:

- **A** — Alerts, Audit Logs, Autofix
- **B** — Branch Protection Rules
- **C** — Code Scanning & CodeQL
- *(…and more through Z)*

Refer to [devsecops-alphabet-soup.md](devsecops-alphabet-soup.md) for the full guide.
